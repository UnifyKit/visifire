Follow the steps required for setting up an environment for compiling and running Visifire source

1) Install Visual studio 2008 with SP1
2) Install Silverlight Tools for Visual Studio 2008 SP1.
3) Go to http://code.msdn.microsoft.com/silverlightut/Release/ProjectReleases.aspx?ReleaseId=1544
4) Click and download Visual Studio Templates.zip file.
5) Once downloaded, open the folder Visual Studio Templates.
6) Copy SilverlightTestProject_CSharp.zip into your "%userprofile%\Documents\Visual Studio 2008\Templates\ProjectTemplates".
7) Copy SilverlightTestClass.zip into your into your "%userprofile%\Documents\Visual Studio 2008\Templates\ItemTemplates".
8) Go to http://code.msdn.microsoft.com/silverlightut/Release/ProjectReleases.aspx?ReleaseId=1544
9) Click and download Microsoft.Silverlight.Testing.zip file.
10) Once downloaded, open the folder Microsoft.Silverlight.Testing.zip and extract it.
11) Open the extracted folder and copy the assemblies into your '%ProgramFiles%\Microsoft SDKs\Silverlight\v2.0\Libraries\Client\' directory for Silverlight 2.0 or '%ProgramFiles%\Microsoft SDKs\Silverlight\v3.0\Libraries\Client\' directory for Silverlight 3.0.
12) You are done.
13) Now you can run the source code of Visifire.
